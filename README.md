# 🎨 Color Memory Rush

Color Memory Rush is a fun and challenging browser-based memory game that tests your ability to remember and match colors. Players must quickly identify and recall color patterns as the difficulty increases with each level.

## 🚀 Features

* Interactive color memory gameplay
* Increasing difficulty levels
* Score tracking system
* Responsive and user-friendly interface
* Fast-paced challenge to improve concentration and memory skills
* Lightweight and easy to run in any modern browser

## 🎮 How to Play

1. Start the game.
2. Observe the sequence of colors displayed on the screen.
3. Memorize the order carefully.
4. Reproduce the sequence by clicking the correct colors.
5. Advance through levels by correctly matching the pattern.
6. The game ends when an incorrect color is selected.
